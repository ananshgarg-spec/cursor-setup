# AI-Powered SEO Content Production Research

## Project Overview

This project was completed as part of an AI research assignment to explore how leading SEO professionals use Artificial Intelligence for content production, search optimization, and organic growth. The objective was to collect, organize, and analyze educational content from industry experts using AI-assisted tools such as Claude Code and Codex.

## Research Objective

The goal of this research was to:

* Identify experienced SEO professionals actively sharing practical AI-powered SEO strategies.
* Collect YouTube transcripts using transcript APIs (Supadata).
* Gather recent LinkedIn posts to understand current industry trends.
* Organize all research into a structured GitHub repository for future reference.
